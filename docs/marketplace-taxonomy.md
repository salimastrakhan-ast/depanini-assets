## 6. Модель объявления

- `type`: `job` (вакансия) | `service` (услуга)
- `category`: id из таблиц выше
- `title`, `description`
- `price` (DZD) + `priceUnit`: fixed / hour / day / month / negotiable
- `wilaya` + `city`
- `phone` — контакт для связи
- `photos` — до 8 фото
- `status`: active / closed
